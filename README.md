# Coffee Sales Dashboard Project

This project analyses coffee sales data using Excel to identify trends in revenue, product performance, and customer purchasing behaviour. The goal is to transform raw sales data into meaningful insights through data cleaning, analysis, and dashboard visualisation.

---

## Project Overview

The dataset contains coffee sales transactions which were cleaned, structured, and analysed to create an interactive dashboard. The dashboard provides insights into:

- Total sales performance
- Product category performance
- Customer purchasing trends
- Time-based sales patterns

---

## Tools Used

- Microsoft Excel  
- Pivot Tables  
- Data Cleaning Techniques  
- Data Visualization (Charts & Graphs)  
- Dashboard Design  

---

##  Key Insights

- Identification of top-performing coffee products  
- Monthly/period sales trends  
- Customer purchase behaviour patterns  
- Revenue distribution across products  

---

## Project Objectives

- Clean and organise raw sales data  
- Perform exploratory data analysis  
- Build an interactive Excel dashboard  
- Extract meaningful business insights
